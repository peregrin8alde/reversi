== 背景

現状認識

* アナログゲームの１つにリバーシというものがある。
* 遊ぶためにはゲーム盤や駒が必要である。
* 基本となるルールは存在するが、ローカルルールなどもあり遊ぶ人によって認識が異なる場合がある。


== 目的

PC 上でデジタルゲームとしてリバーシを遊べるようにする。
