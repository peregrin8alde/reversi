= 業務要件

ここでは、以下についてどのように変化するかを説明する。

* 業務一覧（業務内容、担当者など）
* 業務フロー（各業務ステップの流れ）
* 規模（各業務で扱うユーザやデータなどの規模）
* 時期、時間（各業務が行われる時期や必要とする時間）
* 場所（各業務が行われる場所）
* 指標（各業務の良し悪しを評価する指標）
* 範囲（プロジェクトが提供するシステムが扱う業務範囲）
